# Sito web personale di Stefano Capezzone

L'obiettivo è la realizzazione di un sito web personale che descriva il profilo professionale di Stefano Capezzone e costituisca un blog personale per esprimere il suo punto di vista sui temi dell'Innovazione Tecnologica e sui temi attinenti alle sue attività professionali.

Tra le informazioni, viene fornito il CV di Stefano Capezzone per dare un dettaglio sul suo profilo, ma questo non deve essere né pubblicato né reso scaricabile.

Sul sito deve essere pubblicata una bio riassuntiva di media lunghezza, la cui bozza di testo è fornita.
